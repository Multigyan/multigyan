## üîß TECHNICAL DETAILS FOR DEVELOPERS

### Files Modified:

1. **`components/blog/FlexibleTagInput.jsx`**
   - Changed regex from `/[^\w\s-]/g` to `/[^\p{L}\p{N}\s\-_]/gu`
   - Added `\p{L}` for Unicode letters (all languages)
   - Added `\p{N}` for Unicode numbers
   - Added `u` flag for full Unicode support

2. **`app/(dashboard)/dashboard/posts/new/page.js`**
   - Added `countWords()` helper function
   - Added `countCharacters()` helper function
   - Added `<TextCounter>` component with color coding
   - Added `contentWordCount` memoized value
   - Integrated counters in all text fields

### Regex Explanation:

**Old (Broken for Hindi):**
```javascript
/[^\w\s-]/g
// \w only matches: a-z, A-Z, 0-9, _ (NO Hindi!)
```

**New (Works for All Languages):**
```javascript
/[^\p{L}\p{N}\s\-_]/gu
// \p{L} = All Unicode letters (English, Hindi, Arabic, etc.)
// \p{N} = All Unicode numbers
// u flag = Enable Unicode mode
```
