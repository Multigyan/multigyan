# 🚀 5-Minute Quick Start Guide

**Goal:** Get your enhanced blog system running in 5 minutes!

---

## ⚡ Step 1: Install Dependencies (1 minute)

Open your terminal in the project folder and run:

```bash
npm install @tiptap/extension-youtube @tiptap/extension-text-align @tiptap/extension-underline
```

**Wait for it to finish...** ⏳

✅ Done when you see: `added X packages` and no errors

---

## 🎯 Step 2: Start Development Server (30 seconds)

In the same terminal, run:

```bash
npm run dev
```

**Wait for:** `Ready - started server on http://localhost:3000`

✅ Open your browser to: http://localhost:3000

---

## 🔐 Step 3: Login as Admin (30 seconds)

1. Go to login page
2. Enter your admin credentials
3. Click "Sign In"

✅ You should see the dashboard

---

## ✍️ Step 4: Create Your First Post (2 minutes)

### A. Navigate to Post Creation
1. Click "Posts" in sidebar
2. Click "Create New Post" button

### B. Upload Featured Image
1. **Easy Way:** Drag an image from your computer and drop it
2. **Or:** Click "Choose File" and select an image

**Watch for:**
- "Converting to WebP format..." message
- "Image optimized! X% smaller" message
- Green badge showing compression

✅ Image should display with size info

### C. Add Alt Text
In the "Alt Text" field, type:
```
Beautiful sunset over the ocean
```

✅ Required for SEO!

### D. Enter Title
```
My First Blog Post with Amazing Features
```

✅ Character counter shows below

### E. Write Content

Click in the content editor and type:

```
## Welcome to My Blog!

This is my first post using the **enhanced blog system**.

### Features I Love:
- Automatic WebP conversion
- YouTube embeds
- Google Drive images
- Real-time preview

Check out this video:
(Click YouTube icon and paste: https://www.youtube.com/watch?v=dQw4w9WgXcQ)

```

**Try these:**
- Select text and click **B** for bold
- Select text and click *I* for italic
- Click H2 or H3 for headings
- Click image icon to add more images

✅ Content formats as you type

### F. Select Category
1. Click category dropdown
2. Select "Technology" (or any category)

✅ Category selected

### G. Add Tags
Type one of these:
```
blog, technology, tutorial
```
OR
```
#blog #technology #tutorial
```
OR
```
blog technology tutorial
```
(Then press Enter)

✅ Tags appear as badges

---

## 👁️ Step 5: Preview Your Post (30 seconds)

1. Click "Preview Post" button (top right)
2. Check how it looks
3. Close preview

✅ Everything looks perfect!

---

## 🎉 Step 6: Publish! (30 seconds)

1. Scroll down in the right sidebar
2. Click "Publish Now" (or "Save as Draft")

✅ Success message appears!
✅ Redirected to posts list

---

## 🆘 Having Issues?

### Images Not Uploading?
**Check:**
- Is Cloudinary configured? (`.env.local` file)
- Is file under 10MB?
- Is it a valid image format?

**Quick Fix:** Use "Image URL" tab instead

### Google Drive Not Working?
**Check:**
- Is image shared "Anyone with link"?
- Did you wait 30 seconds after sharing?
- Try opening link in incognito window

### Tags Not Adding?
**Remember to:**
- Press Enter after typing
- Use commas between tags
- Or use hashtags with spaces

### Preview Not Showing?
**Make sure:**
- You have a title or content
- Try refreshing the page

## 💡 Pro Tips for Beginners

### 1. Drag & Drop is Your Friend
Instead of clicking "Choose File":
- Just drag images from your desktop
- Drop directly into the Featured Image area
- Or drop directly into the content editor!

### 2. Use Keyboard Shortcuts
- `Ctrl+B` = Bold
- `Ctrl+I` = Italic  
- `Ctrl+Z` = Undo
- Much faster than clicking!

### 3. Preview Often
Click "Preview Post" frequently to:
- Check how it looks
- Catch formatting issues
- See reading time

### 4. Save Drafts
Don't lose your work:
- Click "Save as Draft" often
- Continue working later
- Publish when ready

### 5. Use Google Drive
If you store images in Google Drive:
- Just paste the share link
- System converts automatically
- No need to download and re-upload!
